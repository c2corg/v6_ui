Allows to syntetically indicate the encountered conditions on each part of the route:

  • Location / elevation / orientation : caracteristics of the location: localization, elevation range, orientation, relief (dale, gully...).
  • Soft : depth of soft snow in cm (sinking with the pole).
  • Total : total snow depth in cm. Leave this field empty if the value is unknown.
  • Comment : quality of the snow, rough stratigraphic analysis, ski quality.
